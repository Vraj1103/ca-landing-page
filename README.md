# HETAL J SHAH & Co. — Chartered Accountant Website

A premium, trust-focused website for HETAL J SHAH & Co., chartered accountants in Ahmedabad (Maninagar). Built with Next.js 15 (App Router) and Tailwind CSS.

## Features

- **Hero** — Headline, subline, primary CTA, subtle grid and gradient background with entrance animation
- **Services** — Audit, Tax, Advisory, Compliance cards with hover states
- **Trust block** — Stats (experience, clients, ICAI) and “Why us” bullets
- **About** — Short narrative and team photo placeholder
- **Testimonials** — Three client quotes
- **CTA strip** — Secondary “Get in touch” before contact
- **Social proof** — Credentials (ICAI, ISO)
- **Contact** — Form, address, phone, email, and embedded Google Maps
- **FAQ** — Accordion; **Legal** — Privacy & Disclaimer pages
- **Header / Footer** — Responsive nav, mobile menu, skip link, focus states
- **Design** — Cormorant Garamond + Source Sans 3, 4-color palette, scroll-reveal

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Tech

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Mobile-first, accessible (WCAG-friendly focus, semantic HTML, skip link)
